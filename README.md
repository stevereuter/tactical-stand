# UFO Intercept: Tactical Stand

UFO Intercept: Tactical Stand is a Battleship-style strategy game with a space-combat theme.

This project continues the story from my other game, UFO Intercept. In that first game, humanity failed to stop the alien approach to Earth. Tactical Stand is the follow-up: a final defensive operation, our last stand to prevent annihilation.

I originally wrote this game more than 10 years ago.

## About the Game

- Genre: Turn-based tactical targeting game (Battleship-style)
- Theme: Space warfare and planetary defense
- Platform: Browser (HTML, CSS, JavaScript)

## Story Premise

After the failed interception effort, the alien fleet has reached Earth. Tactical Stand puts you in command of the final line of defense. Every shot matters.

## Project Notes

This repository contains a modernized version of the original project structure for web deployment.
